# fs-coalesce

Simple module which will return or read the first file to exist in an array of paths.

## Getting Started
Install the module with: `npm install fs-coalesce`

```javascript
var fs_coalesce = require('fs-coalesce');
fs_coalesce.awesome(); // "awesome"
```
